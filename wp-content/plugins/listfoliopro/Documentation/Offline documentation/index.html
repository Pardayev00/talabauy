plugin Updates

There are 3 methods to update WordPress plugins to the newer version. These new updates are essential if you want to have a bug free website as we give new features and improvements to the plugin over time. Some of these updates might be needed in order to make the plugin compatible with WordPress newer versions. There is no risk of losing your settings and pages (If you don't edited plugin core files). However if you had done modifications in parent plugin files, then they will be lost.

Please note that before updating a plugin, we recommend you to make a backup of your modified files. Also, we recommend to deactivate its current version and then upload the zip file with new plugin version then activate and update the plugin.

Update Through WordPress Admin Panel

  1. Navigate to Appearance > plugins Activate another plugin to deactivate the listfoliopro plugin
  2. Delete the installed listfoliopro plugin.
  3. Unzip the file you just downloaded and locate the WordPress plugin listfoliopro.zip
  4. Still within the plugins section, on the header tab, click on “Install plugins” , then on the header second tab click “Upload” .
  5. Click “Browse…”and locate the new plugin file listfoliopro.zip
  6. The “Upgrade existing plugin?” option choose “Yes” from the dropdown list.
  7. Click “Install Now”.
  8. Click “Activate”.

Using FTP

  1. Install an FTP program (Filezilla is free)
  2. Retrieve your FTP login information from your hosting admin panel and login into your hosting via FTP
  3. Unzip the item main file
  4. Locate the listfoliopro.zip file and unzip it. Make sure your unzipping software does not create an extra directory with the same name nested into each other.
  5. Drag and drop plugin main folder into your FTP program with the destination: “YOUR_HOSTING_ROOT/YOUR_SUBDIRECTORY/wp-content/plugins/”
  6. FTP program will give a warning that there is an existing with the same directory name, choose replace all.And wait until all files are replaced
